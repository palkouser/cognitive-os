# ADR-0009: Apache-2.0 license

- Status: Accepted
- Date: 2026-07-13
- Decision owners: Viktor Palkovics

## Context

The project builds on Apache-2.0 licensed LightAgent and may accept external contributions.

## Decision

License Cognitive OS under Apache License 2.0 and maintain NOTICE, third-party notices,
and commit-level donor provenance.

## Alternatives considered

MIT, GPL-family licenses, or a proprietary license.

## Consequences

The license is compatible with the foundation and includes an explicit patent grant;
attribution and license audits remain mandatory.

## Verification

Release checks validate LICENSE, NOTICE, third-party notices, and donor registry entries.
