# ADR 0028: Minimal acceptance before the Verifier Registry

Status: Accepted

Sprint 6 completion uses deterministic structural checks only: JSON Schema validation,
artifact existence and integrity, required-step completion, successful tool terminal state,
and plan consistency. Model judgement and model confidence are never authoritative.

Manual and domain-verifier criteria are explicitly unverifiable. A required unverifiable
criterion blocks completion. The general Verifier Registry and coding, mathematics, logic,
physics, benchmark, and acceptance-policy implementations are deferred to Sprint 7.
